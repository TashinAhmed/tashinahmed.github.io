Automatic Signboard Detection

Automatic Signboard Detection from Natural Scene Image in Context of Bangladesh Google Street View In this research, we propose a segmentation based learning approach with smartimage pre-processing techniques and smart hyperparameter tuning. We have shown comparison among different object detection techniques and displayed the performance behavior of various model related choices. We have also provided a novel dataset BSVSO that has been manually collected and annotated from Google Street View. Future work should aim at extracting information from the detected signboards where cropped signboard image resolution is comparatively low. Target conference: ICRAIR 2020

Rice Grain Disease Detection

Rice grain disease identification using dual phase convolutional neural network-based system aimed at small dataset CNN based dual phase method has been proposed which can work effectively on small rice grain disease dataset with heterogeneity. At the first phase, Faster RCNN method is applied for cropping out the significant portion (rice grain) from the image. This initial phase results in a secondary dataset of rice grains devoid of heterogeneous background. Disease classification is performed on such derived and simplified samples using CNN architecture. Comparison of the dual phase approach with straight forward application of CNN on the small grain dataset shows the effectiveness of the proposed method which provides a 5 fold cross validation accuracy of 88.07%. Submitted for peer review. Journal: Computers and electronics in Agriculture

Cloud Satellite Image Classification

Classification and understanding of cloud structures via satellite images using EfficientNet In this paper,we classified cloud organization patterns using a new scaled up version of Convolutional Neural Network (CNN) named as EfficientNet by Google Research, using satellite data of clouds collected from Kaggle competition which will help us understand how clouds will shape our future climate. We have used dice co-efficient as our final evaluation metric which gave us the score of 65.9% and 64.5% for public and private leaderboard on Kaggle competition respectively. Submitted for peer review. Conference: ICIVC 2020
